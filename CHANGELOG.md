# Changelog

All notable changes to this project will be documented in this file.

- Add `rrfn` generalized RRF score function
- Keep `rrf` / `rrf3` as compatibility wrappers
- Add v0.0.3 SQL and upgrade script

- Add v0.0.2 SQL and upgrade script

## v0.0.1

- Initial MVP release
- `rrf` and `rrf3` scalar functions with NULL-safe behavior
- Docker-based build/test via `make test`
